INTRODUCIR CALIFICACIÓN:

  • ¿Cuál es la diferencia entre Internet y la World Wide Web?

    Internet es una red de redes que conecta a millones de computadoras en todo el mundo. La World Wide Web es un servicio que se ejecuta en Internet y que permite a los usuarios acceder a la información almacenada en las computadoras conectadas a Internet.

  • ¿Cuáles son las partes de un URL?

    El URL (Uniform Resource Locator) es la dirección de un sitio web. Está compuesto por tres partes: el protocolo, el nombre del servidor y el nombre del archivo.

  • ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

    GET: Se utiliza cuando se quiere adquirir un archivo o recurso que se encuentra en un servidor web. HEAD: Similiar a get, usando solamente metadatos de un recurso o archivo. POST: Se utiliza para enviar información a un servidor. PUT: Carga un nuevo recurso al servidor. DELETE: Solicita al servidor web que se borre un recurso en específico

  • ¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

    Se utiliza el método submit que se encarga de enviar el formulario, posteriormente se usa reset que reestablece el formulario a los valores por defecto

  • ¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

    Se utiliza el método GET

  • Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

    La respuesta 200 indica que la solicitud ha tenido éxito y no ocurrió ningún error

  • ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

    Si, es responsabilidad del desarrollador corregir el error 404, ya que es un error que se encuentra en el servidor web

  • ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

    Sí, es responsabilidad del desarrollador, ya que el error 500 se da del lado del servidor

  • ¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.

    Los atributos desaprobados son aquellos que ya no se utilizan en HTML5, por ejemplo el atributo align, acronym, applet, basefont y center

  • ¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

    HTML5 es más semántico, es decir, se enfoca en la estructura de la página web, mientras que HTML4 se enfoca en la presentación de la página web. HTML5 es más fácil de usar y es más compatible con dispositivos móviles

  • ¿Qué componentes de estructura y estilo tiene una tabla?

    Las tablas tienen componentes de estructura como son: caption, colgroup, col, thead, tbody, tfoot, tr, th, td. Y componentes de estilo como son: table-layout, border-collapse, border-spacing, empty-cells, caption-side, border, background, width, height, padding, border, margin

  • ¿Cuáles son los principales controles de una forma HTML5?

    Los principales controles de una forma HTML5 son: input, textarea, select, button, output, datalist, keygen, output, meter, progress

  • ¿Qué tanto soporte HTML5 tiene el navegador que utilizas?

    El navegador que utilizo es Google Chrome y tiene un 100% de soporte HTML5
